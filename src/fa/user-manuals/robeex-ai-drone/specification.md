# ربات پرنده هوشمند برنامه پذیر - RoBeeX

![RoBeeX Drone](/assets/robeex_01.png)

RoBeeX یک پهپاد آموزشی قابل برنامه ریزی است که برای آموزش مفاهیم کدنویسی، رباتیک و هوش مصنوعی از طریق یادگیری عملی طراحی شده است. این پهپاد از زبان‌هایی مانند Python، Arduino و Blockly پشتیبانی می‌کند و دسترسی به آن را برای مبتدیان و کاربران پیشرفته فراهم می‌کند. با سنسورها و دوربین داخلی، امکان انجام پروژه‌ها در زمینه بینایی کامپیوتر، پرواز خودکار و آموزش علوم، فناوری، مهندسی و ریاضی (STEM) را فراهم می‌کند.

::: warning
این کوادکوپتر برای پرواز در محیط داخلی طراحی شده است.
:::

| پارامتر | مقدار |
| -------- | -------- |
| نوع | کوادکوپتر |
| زمان پرواز | تا 8 دقیقه |
| وزن کوادکوپتر | 90 گرم |
| ابعاد | 18 x 18 x 40 میلی‌متر |
| نوع موتور | DC بدون هسته |
| باتری | LiPo 3.8V 1S 1100mAh |
| حداکثر برد پرواز | 50 متر |
| حداکثر سرعت باد مجاز | تا 5 متر بر ثانیه |
| دوربین | وضوح تصویر 3 مگاپیکسل: تا 1024x768 |
| فرکانس | WiFi 2.4GHz |
