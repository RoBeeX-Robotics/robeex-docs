## Edit the Demo program

::: warning
By Default Demo Programs are not editable and you should Duplicate them to be able to make modification to them
:::

## Edit / Duplicate the Demo program

![Duplicate a program](/assets/blockly-programming/duplicate-button.png)

1. Click on the top-right Option button
2. Select `Duplicate`
3. Enter name of the new program
4. Go back to the `Blockly Program Manager` and select the new program to modify it !
