# üì≤Install RoBeeX Application

The **RoBeeX App** is the essential command center for controlling and managing your RoBeeX drone. 

It is used for the following key functions:

* **Activating** your drone after initial setup.
* **Updating firmware** for the drone.
* **Flying and controlling the camera** (photo/video capture).
* **Calibrating sensors** (Accelerometer and Gyro).
* **Programming with Blockly** (Scratch-based coding).
* **Configuring aircraft settings** and flight parameters.
* **Viewing, managing, and saving** captured photos and videos.

### Installation Steps

1.  Download the `.apk` file from the link above.
2.  Open your **Downloads** folder and tap the file to begin the installation.
3.  If prompted, you may need to navigate to your phone settings and allow installation from **unknown sources** to proceed.
4.  Launch the **RoBeeX App** after the installation is complete.
