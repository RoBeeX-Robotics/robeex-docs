  # RoBeeX AI Drone

![RoBeeX Drone](/assets/robeex_01.png)


RoBeeX is a programmable educational drone designed to teach coding, robotics, and AI concepts through hands-on learning. It supports languages like Python, Arduino, and Blockly, making it accessible for beginners and advanced users alike. With built-in sensors and camera, it enables projects in computer vision, autonomous flight, and STEM education.

::: warning
This quadcopter is designed for indoor flight.
:::

| Parameter| Value | 
| -------- | -------- |
| Type| Quadcopter | 
| Flight Time | up to 8 minutes | 
| Quadcopter weight| 90 g |
| Dimensions | 18 * 18 * 40 mm |
| Motor Type | Coreless DC |
| Battery  | LiPo 3.8V 1S 1100mAh |
| Maximum flight range | 50 m |
| Permissible wind speed | up to 5 m/s |
| Camera | 3MP Video resolution: up to 1024x768 |
| Frequency | WiFi 2.4GHz |

## Features 
- Manaual Flight using Smart Phone / PC
- Using Joy Sticks (Connected to PC)
- Programming with Blockly
    - Persision Flight
    - Reading from sensors (IMU, Bottom Range Finder)
    - RGB LED / Bottom Flash Light
    - Camera
        - Color Detection
        - Face Detection
- Pogramming with Python
    - Full control over drone functionlities (same as blockly)
    - Integration with OpenCV
    - Integration with Vision AI Models
